Exact Audio Copy Configuration

  1. You must have the "freedb Metadata Plugin" installed to use the proxy with Exact Audio Copy, the built in freedb will not work.

    To install this, check the box next to it during installation.

  2. Once you have confirmed the "freedb Metadata Plugin" is installed, access the metadata options through the "EAC → Metadata Options" menu.

  3. In the "Metadata Provider" tab, select "freedb Metadata Plugin" from the provider list.

  4. Click "Show options of the selected metadata provider".

  5. Put the server link in the "Use freedb server" text box.

  6. Close the window and click "OK".

  7. To make it so CDs automatically get tagged using the freedb server, access the EAC options through the "EAC → EAC Options" menu.

  8. In the "General" tab, check "On unknown CDs," and select the "automatically access online metadata databse" option.

  9. Setup is complete, insert a CD to test!
